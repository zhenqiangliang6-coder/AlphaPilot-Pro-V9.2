AlphaPilot Pro V9.2 → V10.0 升级路线图
 总目标:​V10.0 = AI 直接做交易决策
V10.0 的核心是:​AI 不再预测涨跌,​而是直接输出买入 / 卖出 / 加仓 / 减仓 / 空仓动作。​系统从“信号驱动”升级为“策略驱动”。​

 升级路线图总览
版本	核心成果	难度	是否需要改架构
V9.2(当前)​	事件驱动 + 双策略 + 风控体系		
V9.3	AI 预测模块		
V9.4	多因子融合(AI + 技术因子 + 量比)​		
V10.0	强化学习策略(RL)​直接做决策		(轻微)​
V10.1-V10.6	真实交易成本、​滑点、​短线奖励 shaping		

 第一阶段:​V9.4 多因子融合(为 RL 做准备)​
 目标
让 AI 的输入更丰富,​让 RL 有更好的“世界模型”。​
 任务
加入更多因子:​量比、​换手率、​分时强弱、​大盘联动、​板块热度、​资金流向
统一因子格式 → 生成 1×N 因子向量
升级信号文件格式(兼容旧版)​
{
  "code": "600821.SH",
  "factors": {
    "volume_ratio": 3.8,
    "turnover": 0.023,
    "market_strength": 0.7,
    "sector_heat": 0.82
  }
}

 第二阶段:​V10.0 强化学习策略(核心版本)​
 定义
AI 直接输出动作:​
0 = 不动
1 = 买入
2 = 卖出
3 = 加仓
4 = 减仓
 新增模块(兼容现有架构)​
env/
 ├── state_builder.py      # 构建状态向量
 ├── reward_engine.py      # 奖励函数(短线风格)​
 ├── simulator.py          # 仿真撮合(含滑点、​手续费)​
 └── env.py                # 主环境

strategies/
 └── rl_strategy.py        # RL 策略模块

train_rl_agent.py          # 训练脚本

 第三阶段:​V10.1-V10.6(短线风格强化学习优化)​
V10.1:​加入真实交易成本
手续费
印花税
滑点(0.02%-0.08%)​
T+1 限制
V10.2:​短线奖励 shaping(3-4 天持仓风格)​
持仓超过 4 天 → 惩罚
盈利超过 2% → 奖励加倍
亏损超过 -1.2% → 惩罚加倍
快速止盈(1-3 天)​→ 奖励加成
V10.3:​RL 风控(学习你的止损体系)​
-0.5% 进入观察
-1.2% 减半
-2.5% 清仓
V10.4:​RL 大盘联动
大盘弱 → 不买
大盘强 → 放宽阈值
下午量比更严格
V10.5:​多股票并行训练(Portfolio RL)​
单股票 → 多股票
单仓位 → 多仓位
V10.6:​实盘级 RL 策略(最终版)​
3-4 天持仓
高频但不乱交易
止损果断
止盈灵活
资金利用率高
交易成本低
适应震荡行情
