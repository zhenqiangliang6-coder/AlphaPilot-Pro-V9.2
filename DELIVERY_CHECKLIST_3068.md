### 4.2 验证要点

**止损日志**：
```
[止损-一级] SZSE.300444 (30/68特殊板块) ... 阈值:1.60%
[止损-一级] SHSE.600000 (普通股票) ... 阈值:1.20%
```

**止盈日志**：
```
[止盈-快速] SZSE.300444 (30/68特殊板块) ... 上限:17.00%
[止盈-快速] SHSE.600000 (普通股票) ... 上限:8.50%
```

### 4.3 常见问题

**Q1: 如何确认30/68股票使用了新策略？**  
A: 查看日志中的"(30/68特殊板块)"标识和对应的阈值参数。

**Q2: 想临时禁用差异化策略怎么办？**  
A: 在`config/settings.py`中注释掉`_3068`后缀的参数，系统会自动回退到默认值。

**Q3: 如何调整30/68股票的止损阈值？**  
A: 修改`config/settings.py`中的`STOP_LOSS_LEVEL1_THRESHOLD_3068`和`STOP_LOSS_LEVEL2_THRESHOLD_3068`。

**Q4: 测试失败怎么办？**  
A: 运行`python test_3068_differentiated_strategy.py`查看详细错误信息，对照`QUICK_REFERENCE_3068_STRATEGY.md`的问题排查章节。
