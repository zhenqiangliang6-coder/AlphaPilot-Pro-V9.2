## 🎯 技术亮点

### 1. 智能环境管理

```batch
:: 自动检测Python
where python >nul 2>&1
if %errorlevel% neq 0 (
    echo ❌ 错误: 未找到Python
    exit /b 1
)

:: 自动创建虚拟环境
if not exist "quant_env\Scripts\python.exe" (
    python -m venv quant_env
)

:: 自动安装依赖
python -c "import gm" 2>nul
if %errorlevel% neq 0 (
    pip install gm -i https://pypi.tuna.tsinghua.edu.cn/simple
)
```

### 2. 灵活的配置选项

```batch
:: 用户可选择是否清空历史
set /p CLEAR_HISTORY="是否清空同步历史? (y/n, 默认n): "
if /i "%CLEAR_HISTORY%"=="y" (
    del "signals\.sync_history.json"
)
```

### 3. 多进程并行启动

```batch
:: 启动3个独立窗口,每个设置不同的环境变量
start "🅰️ 账户A" cmd /k "set VOLUME_RATIO_THRESHOLD=2.0 && python lazy_replay_all.py"
start "🅱️ 账户B" cmd /k "set VOLUME_RATIO_THRESHOLD=3.0 && python lazy_replay_all.py"
start "🅲 账户C" cmd /k "set VOLUME_RATIO_THRESHOLD=4.0 && python lazy_replay_all.py"
```

## ⚠️ 注意事项

### 1. 掘金终端连接

确保掘金终端已登录并运行,否则策略无法执行下单操作。

### 2. 仿真账户准备

使用多账户模式前,需在掘金终端创建3个不同的仿真账户。

### 3. 只读观察模式

正式实盘前,建议开启只读模式:

编辑 `config/settings.py`:
```python
READ_ONLY_MODE = True  # 只记录日志,不下单
TEST_SAFE_MODE = True  # 限制下单金额
```
